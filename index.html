Hello World

Welcome to my blog! I've been meaning to start writing for a while now, and I finally decided to just do it. No fancy frameworks, no complicated build tools — just plain HTML and CSS hosted for free on GitHub Pages.

I'm based in Bozeman, Montana, and I plan to write about whatever catches my interest: tech, life in the mountains, projects I'm working on, things I'm learning. Nothing too serious, nothing too polished — just honest thoughts put out into the world.

Why keep it simple?

There are a million blogging platforms out there, but most of them come with strings attached — subscriptions, algorithms, ads, lock-in. A static HTML site is:

  • Free forever — GitHub Pages costs nothing
  • Fast — no database, no server-side rendering, just files
  • Indestructible — it'll work as long as the web exists
  • Totally mine — no platform can take it away

If you're reading this, thanks for stopping by. More posts coming soon.
